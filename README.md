🎮 CS:GO Инвентарь Конфигуратор / Inventory WebUI for csgo_gc

    Просто открой в браузере, накрути себе скинов, агентов и перчаток — и сгенерируй готовый inventory.txt для своего сервера csgo_gc.
    Никаких серверов, никакой магии — чистый HTML/JS.

https://via.placeholder.com/800x400.png?text=CS%253AGO+Inventory+Configurator

CS:GO Inventory Configurator — это локальный веб-инструмент, который помогает собирать инвентарь для эмулятора/мода csgo_gc.
Вы просто выбираете предметы (оружие со скинами, агентов, перчатки), настраиваете их параметры (finish catalog, паттерн, флоат) и получаете готовый файл inventory.txt, готовый к загрузке в папку csgo_gc.
Вдохновлён необходимостью быстро конфигурировать предметы без правки текстовика руками.
✨ Что умеет эта штука

    🔫 Добавлять скины на оружие
    Выбор оружия (def index), качество, редкость, finish catalog, паттерн, флоат.

    🕵️ Добавлять агентов
    Все агенты CT и T с их официальными ID.

    🧤 Добавлять перчатки
    Выбор типа перчаток, finish, паттерн, флоат.

    🔖 Наклейки и 📦 Кейсы — пока в разработке (заглушки).

    🗑️ Удалять предметы из списка перед экспортом.

    📥 Экспорт в inventory.txt в формате KeyValues, который понимает csgo_gc.

    🔢 Указание последнего ID предмета — чтобы новые предметы дописывались с правильными номерами.

    🧠 Встроенная база оружия, качеств, редкостей, агентов, перчаток — ничего дополнительно искать не надо.
