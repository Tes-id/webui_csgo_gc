🎮 CS:GO Инвентарь Конфигуратор / Inventory WebUI for csgo_gc

    Просто открой в браузере, накрути себе скинов, агентов и перчаток — и сгенерируй готовый inventory.txt
![CS:GO Inventory Configurator Preview](https://github.com/Tes-id/webui_csgo_gc/raw/main/preview.png)

CS:GO Inventory Configurator — это локальный веб-инструмент, который помогает собирать инвентарь для csgo_gc.
Вы просто выбираете предметы (оружие со скинами, агентов, перчатки), настраиваете их параметры (finish catalog, паттерн, флоат) и получаете готовый файл inventory.txt, готовый к загрузке в папку csgo_gc.
Вдохновлён необходимостью быстро конфигурировать предметы без правки текстовика руками.

📦 Установка и запуск

Ничего устанавливать не надо!
Просто скачай index.html и открой его в любом современном браузере.
Можно даже запустить с флешки или залить на хостинг — всё равно это статика.
