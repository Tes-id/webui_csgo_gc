🎮 CS:GO Инвентарь Конфигуратор / Inventory WebUI for csgo_gc

    Просто открой в браузере, накрути себе скинов, агентов и перчаток — и сгенерируй готовый inventory.txt для своего сервера csgo_gc.
    Никаких серверов, никакой магии — чистый HTML/JS.

CS:GO Inventory Configurator — это локальный веб-инструмент, который помогает собирать инвентарь для эмулятора/мода csgo_gc.
Вы просто выбираете предметы (оружие со скинами, агентов, перчатки), настраиваете их параметры (finish catalog, паттерн, флоат) и получаете готовый файл inventory.txt, готовый к загрузке в папку csgo_gc.
Вдохновлён необходимостью быстро конфигурировать предметы без правки текстовика руками.
✨ Что умеет эта штука

    🔫 Добавлять скины на оружие
    Выбор оружия (def index), качество, редкость, finish catalog, паттерн, флоат.

    🕵️ Добавлять агентов
    Все агенты CT и T с их официальными ID.

    🧤 Добавлять перчатки
    Выбор типа перчаток, finish, паттерн, флоат.

    🔖 Наклейки и 📦 Кейсы — пока в разработке (заглушки).

    🗑️ Удалять предметы из списка перед экспортом.

    📥 Экспорт в inventory.txt в формате KeyValues, который понимает csgo_gc.

    🔢 Указание последнего ID предмета — чтобы новые предметы дописывались с правильными номерами.

    🧠 Встроенная база оружия, качеств, редкостей, агентов, перчаток — ничего дополнительно искать не надо.

🚀 Как пользоваться

    Скачай index.html (или открой его прямо в браузере — всё работает локально).

    Узнай последний ID предмета в твоём текущем inventory.txt (находится в папке csgo_gc). Впиши его в поле Last item ID справа.

    Выбери тип предмета (скин, агент, перчатки) сверху.

    Настрой параметры:

        Для скина: оружие, качество, редкость, Finish catalog (берётся с сайта типа csgoskins.gg), паттерн, флоат.

        Для агента: просто выбери нужного.

        Для перчаток: тип, finish, паттерн, флоат.

    Нажми «Добавить ...» — предмет появится в списке справа.

    Повторяй, пока не соберёшь нужный инвентарь.

    Нажми «Экспорт в inventory.txt» — браузер скачает готовый файл.

    Замени или добавь содержимое этого файла в твой inventory.txt в папке csgo_gc.

📁 Формат экспортируемого файла

Файл генерируется в формате, который использует csgo_gc:
json

"items"
{
	"21"                   // порядковый номер (lastId + 1)
	{
		"inventory"		"1"
		"def_index"		"7"          // weapon_ak47
		"level"		"1"
		"quality"		"3"          // ★
		"flags"		"0"
		"origin"		"8"
		"in_use"		"0"
		"rarity"		"4"          // Restricted
		"attributes"
		{
			"6"		"105.000000"   // finish catalog
			"7"		"592.000000"   // pattern
			"8"		"0.123456"     // float
		}
		"equipped_state"
		{
		}
	}
	...
}

Для агентов и перчаток аттрибуты (finish, pattern, float) добавляются только там, где они нужны (у агентов их нет).
🧪 Известные баги (и почему это всё равно работает)

    Иконки старых предметов иногда не обновляются — но это проблема клиента, мы пофиксили костылём.

    Редкость у некоторых типов предметов может отображаться неверно — да мне пофиг, веришь? Главное, что в инвентаре работает.
